从地质学走向AI时代的跨界探索者

我叫奚成朋，从中国地质大学到南京大学，从地质勘探到AI项目管理，我用十年时间完成了一次看似不可能的跨界转型。我的故事，是关于如何用跨学科思维在科技行业找到属于自己的独特位置。

本科阶段，我在中国地质大学（武汉）系统学习了资源勘查工程专业，GPA达到3.8/4.0，专业排名前10%，连续获得校级一等奖学金并荣获优秀毕业生称号。这段经历培养了我严谨的数据思维、强大的逻辑分析能力，以及面对复杂问题时抽丝剥茧的系统性思考方式。地质学教会我的不只是专业知识，更是一种"在不确定性中寻找规律"的能力，这种能力在我后来的AI数据工作中发挥了至关重要的作用。

硕士阶段，我进入南京大学攻读地质工程（AI应用方向）硕士学位，主攻机器学习与数据挖掘方向。期间，我深度学习了机器学习、数据挖掘、Python数据分析、深度学习、自然语言处理等核心课程，并将AI技术应用于地质研究，发表了2篇高质量学术论文，参与了国家自然科学基金项目《深层矿产资源智能评价系统》。这段学术训练，让我真正掌握了从数据中提取价值、构建智能化系统的核心能力。

从腾讯技术工程事业群到网易互娱，再到蚂蚁集团，我的职业轨迹始终围绕AI数据治理与项目管理这一核心主线。在腾讯TEG的三年里，我主导了AIdata/Qlabel平台迭代、AI数据安全联合项目（成本降低40%）、云度平台迁移（170+指标、20+看板）等公司级重点项目。在网易互娱，我接触了游戏行业技术工具体系与风控合规，拓宽了业务视野。

如今，我在蚂蚁集团担任AI项目经理，主要负责支小宝AI生活管家与AQ健康管家两大核心产品，以及国家级保密AI项目。推动大模型准确率从60%提升至88%、医疗AI识别达行业领先水平。在蚂蚁，我正在书写职业生涯的新篇章。

《基于全卷积神经网络拾取地震波时间窗口的研究》
南京大学 · 硕士电子期刊 · 2022年（第05期）
研究背景：利用地震波形资料反演地球内部结构是地震学成像领域最先进的反演方法之一，但传统时窗拾取方法已不能满足海量数据反演时的效率要求。

研究方法：采用全卷积神经网络自动拾取复杂地震波信号，从实际勘探地震数据中选取37960条波形样本，每个样本重采样为4992个点。搭建36个卷积层（含7个降采样和7个上采样部分），经1000次迭代训练后达到较好识别效果。

核心成果：预测每条地震记录约需10-5秒，相比传统方法（STA/LTA）快约10000倍，为百万级勘探地震数据的时间窗口选取提供高效技术支撑。
